

## Run Locally

**Prerequisites:**  Node.js


1.  **Install dependencies:**
    `npm install`
2.  **Run the app:**
    `npm run dev`

**API Key Configuration:**

This application is configured to use ambient authentication (e.g., `window.aistudio`) for the Gemini API and does not require a `.env.local` file for local development. This is a more secure practice than exposing keys in environment files.
